# cmake-template
CMake template for C++ project with external dependency from GitHub.
ZeroMQ (libzmq) as an external dependency example.
It supports rtags. More details in build.sh script.

Build:
----
* `$  cd cmake-template`
* `$  mkdir build`
* `$  cd build`
* `$  cmake ..`
* `$  make -j3`

or just:
----

* `$  cd cmake-template`
* `$  bash build.sh`
